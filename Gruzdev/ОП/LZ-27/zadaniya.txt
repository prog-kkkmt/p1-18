1. Упражнение 3.2. Что такое стандартный конструктор?
	Конструктор - это специальная функция-член, которая позволяет инициализировать объекты данного класса. А стандартный конструктор - это конструктор, который выполняется автоматически, если осуществляющий инициализацию объекта конструктор не вызван явно в коде. Стандартный конструктор используется независимо от то того, где конкретно определена переменная.

2. Упражнение 3.3. Назовите три способа инициализации объекта класса string.
	1 способ:	string s1(s); // Инициализая строки s1 как копии строки s
	2 способ:	string s2("value"); // Инициализация строки s2 как копии строкового литерала
	3 способ:	string s3(n, 'c'); // Инициализация строки s2 последовательностью n из символов c
