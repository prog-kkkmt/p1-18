Задача 1. (Составитель: Л.Гусятинер.  09.05.2006, Ковров, КГТА, ПМ и САПР)
Создать стек( магазин ) для хранения патронов
определенного диаметра( от min до max ). О каждом известны: диаметр,
идентификационный номер( для каждого - уникальный ). Имеются также две
коробочки: с патронами разного диаметра и пустая, обе организованные в виде
текстовых файлов. box1.txt, box2.txt. Пределы диаметров и емкость магазина
хранятся в файле magazine.txt. Известно общее количество патронов в первой
коробочке и емкость магазина. Патрон произвольным образом берется из
коробочки и, если подходит по размерам, вставляется в магазин. Если нет -
кладется в другую коробочку. После того, как магазин полон, либо коробка пуста,
он "выстреливается" в пустой файл. Если коробка не пуста, операция повторяется.
  В результате работы должны получиться 4 файла:
   - остатки в 1-й коробочке;
   - остатки во 2-й коробочке;
   - отстрелянные патроны;
   - подходящие размеры

Задача 2. (Составитель: Л.Гусятинер.  09.05.2006, Ковров, КГТА, ПМ и САПР)
Рассмотрим язык СтекЛО. ( STACK Language Ordinary ). Язык имеет 5 команд:
- var ( Объявление переменной ) - унарная
- let ( Присваивание значения ) - бинарная
- push( Втолкнуть ) - унарная
- pop( Вытолкнуть ) - унарная
- print( Печаать ) - унарная
В строке размещается только одна команда (с любой позиции). Могут быть пустые
  строки.
Возможны 4 вида ошибок
- переменная не объявлена;
- переменная не инициализирована;
- переполнение стека( при попытке выполнить команду push );
- стек пуст( при попытке выполнить команду pop ).

Имеется файл in.txt, хранящий строки
программы работы со стеком. stack - системная переменная, значение определяет
размер стека.
  Программа создает файлы Out.txt с результатами выполнения программы, trace.txt
( протокол работы )

In.txt:
==========
 var a
 var b
 var c
 var d
 var e
 let stack 10
 let a 5
 let b 6
 let c 7
 push a
 push b
 push c
 pop c
 pop d
 push c
 pop d
 print d
 print e

Out.txt
==========
d=7
var e not defined

Trace.txt
==========
stack=[]

c=7

stack=7,6,5
stack=6,5
c=7
stack=5
d=6
stack=7,5
stack=5
d=7
d=7
var e not defined

Ожидаемый результат
Файлы с исходным кодом программ на гитхаб.
