Упражнения раздела 2.5

1) Упражнение 2.24. Какие из следующих определений (если они есть) недопустимы? Почему? Как их исправить?
(a) int ival = 1.01;
(b) int &rvall = 1.01;
(с) int &rval2 = ival;
(d) const int &rval3 = 1;

Ответ: Неправильный вариант b) так как инициализировать можно только константные ссылки

2) Упражнение 2.25. Какие из следующих присвоений (если они есть) являются недопустимыми 
(с учетом предыдущих определений)? Если таковые есть, объясните, почему.
(а) rva12 = 3.14159;
(b) rva12 = rva13;
(с) ival = rval3;
(d) rval3 = ival;

Ответ: d) не правильный так как выше написано что rval3 это константно ссылочный тип и его не льзя инициализировать

3) Упражнение 2.26. В чем разница между определениями (а) и присвоениями (Ь)? 
Какие из них (если есть) недопустимы?
(a)	int ival = 0;	(b) ival = ri;
const int &ri =0;	ri = ival;

Ответ: b) не допустим ri это константный ссылочный тип. В варианте (а) мы создаем переменные, а в (b) мы инициализируем
