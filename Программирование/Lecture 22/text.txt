Работа с файлами в стиле c++ 
Библиотека fstream обеспечивает работу с файлами.
Она очень похоже на библиотеку iostream.
Только вместо стандартного ввода и вывода,
здесь идет  ввод/вывод через файлы.

Чтобы открыть некоторый файл на чтение 
определяеться переменная ifstream %имяпеременной%("%имя файла с путем или без него%")
После этого переменную можно использовать как "cin"
ifstream input (" input .txt");
input >> name ;

Аналогично происходит работа с теми файлами в которые мы хотим что-то записать
Для того чтобы открыть такой файл  надо создать переменную ofstream %имяпеременной%("%имя файла с путем или без него%")
После этого переменную можно использовать как "cout"
ofstream output (" output .txt");
output << "Hi , " << name << endl ;

Файлы открываються в момент определения переменной,
а закрываються в тот момент когда переменная перестает существовать,
то есть при выходе из соответствующей функции 
или соответствующей области видимости
